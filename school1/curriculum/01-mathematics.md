# Mathematics Curriculum

Mathematics is the main dependency for physics, control, AI, graphics, signal processing, and chip design. Build it for fluency, not credentialism.

## Stage 1: Algebra, Geometry, Trigonometry, Precalculus

Objectives:

1. Manipulate equations quickly and correctly
2. Understand functions, graphs, transformations, and inverses
3. Be comfortable with trigonometric identities and radians
4. Use vectors, coordinate geometry, and basic complex numbers
5. Perform dimensional and scale reasoning confidently

Outputs:

1. Function sketch notebook
2. Trig identity drill sheets
3. Unit analysis reference sheet
4. Rust CLI for function evaluation and plotting-ready CSV generation

## Stage 2: Calculus I and II

Topics:

1. Limits and continuity
2. Derivatives and interpretation
3. Optimization
4. Integrals and accumulation
5. Series and approximations
6. Multivariable introduction

Outputs:

1. Derivative and integral formula sheets written from memory
2. Optimization exercise sets
3. Rust numerical differentiation and integration tools

## Stage 3: Linear Algebra

Topics:

1. Systems of linear equations
2. Matrix operations
3. Vector spaces and bases
4. Linear transformations
5. Eigenvalues and eigenvectors
6. Orthogonality and least squares

Why it matters:

1. Control systems
2. Computer graphics
3. Robotics kinematics
4. AI / ML
5. State estimation

Outputs:

1. Rust matrix library exercises using `nalgebra`
2. Least squares solver
3. Eigenmode simulation for simple systems

## Stage 4: Differential Equations

Topics:

1. First-order ODEs
2. Second-order linear systems
3. Forced response
4. Stability intuition
5. Laplace transform introduction
6. PDE exposure later as needed

Outputs:

1. Spring-mass-damper solver
2. Predator-prey model
3. Damped oscillator comparison: analytical vs numerical

## Stage 5: Probability, Statistics, and Discrete Math

Topics:

1. Counting and combinatorics
2. Random variables
3. Expectation and variance
4. Conditional probability and Bayes
5. Common distributions
6. Estimation and inference basics
7. Graphs, logic, sets, and proofs

Outputs:

1. Monte Carlo simulation toolkit in Rust
2. Simple Bayesian update exercises
3. Estimation notes for robotics and AI

## Standard Of Mastery

You know a math topic when you can:

1. State the definitions precisely
2. Do a derivation without pattern-matching a memorized trick
3. Solve a new problem
4. Implement the concept computationally
5. Explain the geometry or physical meaning
