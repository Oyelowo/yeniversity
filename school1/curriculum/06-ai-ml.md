# AI And Machine Learning Curriculum

AI should be learned after enough math, probability, and systems intuition are in place. Learn it as optimization, representation, and engineering under constraints.

## Stage 1: Mathematical Foundation Review

Before serious ML, confirm:

1. Linear algebra
2. Multivariable calculus
3. Probability
4. Basic optimization intuition

## Stage 2: Classical Machine Learning

Topics:

1. Linear regression
2. Logistic regression
3. Regularization
4. Trees and ensembles
5. Clustering
6. Evaluation and error analysis

Outputs:

1. Small ML implementations in Rust where practical
2. Data preprocessing notes
3. Evaluation checklist

## Stage 3: Neural Networks

Topics:

1. Perceptrons and MLPs
2. Backpropagation
3. Optimization methods
4. Initialization and normalization intuition
5. Overfitting and generalization

Outputs:

1. Tiny neural network implementation
2. Gradient checking exercises
3. Loss landscape notes

## Stage 4: Vision, Sequence Models, And RL Exposure

Topics:

1. CNN intuition
2. Sequence modeling overview
3. Reinforcement learning basics
4. World models and control links

Outputs:

1. Simple perception experiments
2. Robotics-relevant AI notes
3. Deployment and inference constraints summary

## Stage 5: Systems View

Topics:

1. Data pipelines
2. Compute and memory constraints
3. Quantization and deployment
4. Edge AI in embedded systems

Outputs:

1. Notes linking AI to robots, phones, cameras, and hardware accelerators
2. Mini end-to-end project using sensing plus inference
