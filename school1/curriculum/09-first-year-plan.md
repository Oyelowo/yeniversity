# First-Year Plan

This plan is the concrete execution layer for the broader roadmap.

## Months 1 To 3

Primary focus:

1. Algebra and precalculus cleanup
2. Calculus I
3. Mechanics
4. Rust fundamentals for scientific programming

Outputs:

1. Function and trig notes
2. Mechanics note set
3. At least 30 to 50 solved exercises
4. `projectile_motion.rs`
5. One extra simulation: pendulum or free fall with drag

## Months 4 To 6

Primary focus:

1. Calculus II
2. Linear algebra
3. Intro electricity and magnetism
4. Circuit theory

Outputs:

1. Linear algebra note set
2. 30 to 50 more solved exercises
3. RC / RL simulation tools
4. One matrix computation utility in Rust

## Months 7 To 9

Primary focus:

1. Differential equations
2. Probability and statistics basics
3. Waves and signals introduction
4. Analog electronics introduction

Outputs:

1. Spring-mass-damper simulator
2. Signal generation and filtering code
3. Probability and Monte Carlo notebook or CLI tools
4. Notes connecting ODEs to control and circuits

## Months 10 To 12

Primary focus:

1. Digital logic and computing basics
2. Control fundamentals
3. Robotics foundations
4. AI / ML foundations

Outputs:

1. PID simulation
2. Simple logic simulator or finite-state-machine project
3. Differential-drive robot simulator
4. One end-to-end capstone design note

## End-Of-Year Standard

By the end of year one, the target is:

1. Strong mathematical re-entry completed
2. Mechanics and E and M basics in place
3. Circuit intuition established
4. Rust simulation habit established
5. Several working modeling tools completed
6. Ready to move into embedded systems, control, robotics, and more serious hardware work
