# Project Ladder

Projects are the proof of learning. This ladder is ordered so each build pulls on the theory from the previous phase.

## Phase 1 Projects

1. Projectile motion simulator
2. Pendulum simulator
3. Numerical derivative and integral toolkit
4. Circuit law solver for small resistor networks

## Phase 2 Projects

1. Spring-mass-damper simulator
2. RC, RL, and RLC transient simulator
3. Matrix solver and least-squares fitter
4. Basic signal generator and filter visualizer

## Phase 3 Projects

1. Logic simulator or tiny CPU emulator
2. Motor model and PID controller
3. Sensor data logger and analysis tool
4. Camera geometry or optics simulator

## Phase 4 Projects

1. Differential-drive robot simulator
2. Sensor fusion demo
3. Vision plus control mini-system
4. Embedded hardware prototype

## Phase 5 Projects

Choose at least one:

1. Small drone stack
2. Small EV / vehicle subsystem
3. Custom camera or vision pipeline
4. FPGA or digital design project
5. Fabrication or tooling experiment

## Rule

Each project should produce:

1. A short design note
2. A simulation or code artifact
3. A postmortem documenting what failed and what changed
