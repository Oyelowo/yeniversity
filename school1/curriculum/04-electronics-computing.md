# Electronics, Computing, And Embedded Curriculum

This is the bridge from physical law to useful machines.

## Stage 1: Circuit Theory

Topics:

1. Voltage, current, power, energy
2. Ohm's law
3. Kirchhoff laws
4. Series and parallel networks
5. Thevenin and Norton equivalents
6. Transient response in RC and RL circuits

Outputs:

1. Circuit analysis note set
2. Hand-solved exercise sheets
3. Rust circuit solver toys and plotting data export

## Stage 2: Analog Electronics

Topics:

1. Diodes
2. BJTs and MOSFETs
3. Biasing
4. Amplifiers
5. Op-amps
6. Filters
7. Noise and measurement basics

Outputs:

1. Amplifier design notes
2. Filter calculations and simulations
3. Sensor interface studies

## Stage 3: Digital Logic And Computer Architecture

Topics:

1. Boolean algebra
2. Combinational logic
3. Sequential logic
4. Finite state machines
5. Datapath and control
6. Memory hierarchy basics
7. CPU architecture overview

Outputs:

1. Logic design exercises
2. Simple CPU architecture notes
3. Rust logic simulator or emulator project

## Stage 4: Embedded Systems With Rust

Topics:

1. Rust fundamentals for systems work
2. Ownership, lifetimes, traits, and error handling
3. Numeric programming patterns
4. Microcontroller basics
5. Interrupts, timers, serial communication
6. Sensor readout and control loops

Outputs:

1. Rust study notes for embedded and scientific programming
2. Serial logger utility
3. Sensor processing utilities
4. One embedded project once hardware is ready

## Stage 5: Power, Interfaces, And Instrumentation

Topics:

1. Regulation and power conversion basics
2. ADC and DAC intuition
3. SPI, I2C, UART, CAN overview
4. Measurement and debugging discipline

Outputs:

1. Interface cheat sheets
2. Debugging checklist
3. Hardware bring-up notes
