# Exercise Sheet 03 - Sets, Elements, Subsets, And Operations

## Part A - Element Or Subset

Let:

```text
A = {1, 2, 3}
B = {2, 3}
C = {{2}, 3}
```

Say whether each statement is true or false.

1. `2 ∈ A`
2. `{2} ⊆ A`
3. `{2} ∈ A`
4. `B ⊆ A`
5. `2 ⊆ A`
6. `{2} ∈ C`
7. `2 ∈ C`

## Part B - Compute Set Operations

Let:

```text
X = {1, 2, 3, 4}
Y = {3, 4, 5, 6}
```

Find:

1. `X ∪ Y`
2. `X ∩ Y`
3. `X \ Y`
4. `Y \ X`

## Part C - Explain The Difference

Answer briefly:

1. What is the difference between `x ∈ A` and `{x} ⊆ A`?
2. Why is `{2} ∈ A` false when `2 ∈ A` is true?

## Part D - Mini Reasoning

Let:

```text
M = {a, b}
N = {a, b, c}
```

Answer:

1. Is `M ⊆ N`?
2. Is `N ⊆ M`?
3. Is `M ⊂ N`?
4. Is `M ∩ N = M`?
