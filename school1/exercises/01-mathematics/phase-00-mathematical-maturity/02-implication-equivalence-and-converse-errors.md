# Exercise Sheet 02 - Implication, Equivalence, And Converse Errors

## Part A - Meaning

Let:

- `p`: `The sensor is calibrated.`
- `q`: `The robot can estimate position accurately.`

Write in plain English:

1. `p -> q`
2. `q -> p`
3. `!q -> !p`
4. `p <-> q`

## Part B - Truth Tables

Construct full truth tables for:

1. `p -> q`
2. `q -> p`
3. `p <-> q`

## Part C - Identify The Error

For each pair, decide whether the second statement follows from the first.

1.

```text
If a number is divisible by 4, then it is even.
Therefore, if a number is even, it is divisible by 4.
```

2.

```text
If a device is submerged in water, it may fail.
Therefore, if a device fails, it was submerged in water.
```

3.

```text
If a file is deleted, it is no longer present in the folder.
Therefore, if a file is not present in the folder, it was deleted.
```

## Part D - Contrapositive

Write the contrapositive of each statement.

1. If a number is divisible by 6, then it is divisible by 3.
2. If the circuit is closed, current flows.
3. If the authentication succeeds, the credentials were valid.

## Part E - Equivalence Or Not

Say whether each statement describes implication or equivalence.

1. A number is even if and only if it is divisible by 2.
2. If a process crashes, the program stops executing normally.
3. A shape is a square only if it has four equal sides and four right angles.

## Part F - Reflection

Answer briefly:

1. Why is the converse a common trap?
2. Why is the contrapositive safer than guessing from intuition?
